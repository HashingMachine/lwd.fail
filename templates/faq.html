What is this site?

This site is a list of lightwalletd servers that includes useful information about them. It helps both users of Zcash wallets and developers alike as it provides information regarding the servers' blockchain, height and whether it is online.

This site also helps reduce centralisation by listing all existing lightwalletd servers, allowing users to pick different servers for each Zcash interaction and thus preventing a single server from logging all of their activities, while also avoiding being prevented from using Zcash by offline servers. The list is also available in JSON format for easy integration in applications that want to automate that.

lwd.fail is inspired by monero.fail.

What is lightwalletd?

Lightwalletd is a daemon that provides a bandwidth-efficient interface to the Zcash blockchain for mobile-friendly Zcash wallets. See the repository for detailed information and setup instructions.

I added a server but I can't see it in the list.

The list is updated once every 30 minutes. If the server is not in the list after that time interval and you are sure that you are reading the right list (e.g. mainnet and not testnet or vice-versa), check the offline checkbox and if it still does not appear make sure that you used the right scheme (e.g. https and not http or vice-versa) and the right port (lightwalletd listens on port 9067 by default). If you are sure that everything is correct, it could be that the server was offline on the first status check or that a client-side issue affected reachability, causing the server to be removed. In this case, try adding the server again, and if you are sure that the server is up open an issue on GitHub or contact me.
